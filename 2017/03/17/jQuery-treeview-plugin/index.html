在项目中导入相关的文件

  • 下载文件，treeview开源地址：https://github.com/jzaefferer/jquery-treeview
  • 将treeview导入项目[app]/js/,包括：
    • jquery-1.8.3.js 放在[app]/js/下,treeview依赖jquery库
    • images文件夹 插件相关的图标文件
    • screen.css 屏幕显示相关css
    • jquery.treeview.css 核心css
    • jquery.treeview.min.js 核心js
    • jquery.treeview.async.js 支持异步js
    • jquery.cookie.js 支持缓存js
