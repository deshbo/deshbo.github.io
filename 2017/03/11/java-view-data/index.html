java视图值的使用

开发中,数据库字段如：日期Long,性别int 在页面的显示问题。
视图值: 视图值是一种用于界面显示的变量值,该值不具体对应某个数据库字段,它服务与某个数据库字段。
当数据库中的某个字段值不便于显示时，为该字段添加视图值，用于显示对应的信息。

  1. 在Model中定义一个String类型的变量,变量名是无法合理显示的字段名+View
  2. 为其提供get方法
  3. 在其对应变量的set方法中对这个View值进行初始化

日期类型视图值

性别类型的视图值

  • 在页面上通过 OGNL 或 EL 表达式取值

  • 如果相应的取值太多,可以采用如下方式:

  • 相应的初始化

    public void setGender(Integer gender) {
    this.gender = gender;
    this.genderView = genderMap.get(gender);
}

    同时，在页面标签中可以通过如下方式动态的调用回显:(需要设置genderMap 为public)

    <s:select list="@me.erp.auth.emp.vo.EmpModel@genderMap" name="em.gender"/>
